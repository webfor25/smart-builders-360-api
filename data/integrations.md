# Integrations and External Tools

## Microsoft Teams: cannot give control with 3 monitors
If using Displaylink Manager with three monitors or more:
- You cannot give control of your screen if Displaylink Manager Driver is 1.12.1 (build 54) or less.

### Mac users
- Turn off Displaylink Manager.
- Plug one HDMI directly into the Macbook Pro.
- If using iMac/Macbook, try HDMI → USB-C directly into the Macbook/iMac.
- If it still does not work, remove all external monitors. Teams should work as expected.

### Windows users
- Use an HDMI cable and plug directly into a Windows computer.
- If it still does not work, remove all external monitors. Teams should work as expected.

## PDFs and SignHere documents not displaying company info/logo
- Empty cache in your internet browser.
- Quit the browser.
- Restart the browser.
