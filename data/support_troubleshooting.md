# Smart Builder 360 Support Troubleshooting

## Unresponsive pages
If a page is unresponsive:
1. Empty cache
2. Restart browser
3. Restart computer
4. Try another browser (Microsoft Edge or Safari)
5. Try another computer/device to check if issue persists
6. Check internet speed (congestion may be an issue)
7. Create a new Customer and check the same page:
   - If it is unresponsive again, the original customer record might be corrupt.
   - Download any customer information you can before deleting the corrupt customer.

## Contractor details missing
If Contractor Details are not appearing:
- The customer may have been created before Contractor Details were linked.
- Create a new Customer and Contractor Details should appear.

## Jobs are not being displayed
If your jobs are not displayed but the Admin’s jobs are displayed:
- Check Admin settings and verify “Project Access” is NOT selected.
- If Project Access is selected, only jobs created by that user will be displayed.
- This is intended for teams with multiple sales associates.
