Smart Builder 360 Knowledge Base
1. NOT RECEIVING AN EMAIL: If Contractor has an SBCGlobal or Yahoo email address: We encourage using a Gmail Account. Set it up if need be. If not, go to Yahoo Mail contacts and add a new Contact. Contact email address should be “no-reply@smartbuilder360.com”. You can use your name as the contact. FYI - SBCGlobal email forwards to Yahoo.
2. UNRESPONSIVE PAGES: If a page is unresponsive.
a. Try to empty cache
b. Restart Browser
c. Restart Computer
d. Use another Browser such as Microsoft Edge or Safari
e. Use another computer or device to see if problem exists
f. Check internet speed to see if congestion is an issue.
g. Create a new Customer and see if the same page is unresponsive. If so, the original client with a page that is unresponsive might be corrupt. Create a new customer but download all customer information you can before deleting.
3. CONTRACTOR DETAILS MISSING: If Contractor Details are not appearing….is it because they created a customer BEFORE the Contractor Details were linked? If so….create a new Customer and the Contractor Details will appear
4. NOT RECEIVING SIGNHERE EMAIL: If the sender or recipient does not receive a SignHere document for signature. Try another email address. You will be notified for all future documents to be signed.
5. Can't Open PDF Files via Safari on IOS Device: Go to Settings on IOS device. Search for Safari. Scroll down to Pop Up Blocker…..Toggle Off.
6. Microsoft Teams - Cannot Give Control of your screen when using 3 monitors at once: If you’re using Displaylink Manager with three monitors or more you cannot give control of your screen if you’re using Displaylink Manager Driver 1.12.1 (build 54) or less.
FOR MAC USERS: If you are using a Macbook Pro please turn off Displaylink Manager and only plug one HDMI directly into the Macbook Pro. If you are using an iMac or Macbook try using a HDMI to USB-C and plugin directly into a Macbook or iMac. If this does not work remove all external monitors and Teams should work as expected
FOR WINDOWS USERS: Try using a HDMI cable and plug directly into a Windows computer. If this does not work remove all external monitors and Teams should work as expected.
7. Jobs are not being displayed: If your jobs are not being displayed, but the Admin’s jobs are displayed check the Admin settings page to verify that “Project Access” is not selected. If Project Access is selected, jobs only that user created will be displayed. This is intended for companies that employ several sales associates so only their jobs will be displayed rather than all the Company jobs.
8. PDF’s and SignHere Documents not displaying Company information or logo. Empty cache on internet browser. Quit internet browser. Restart internet browser.
